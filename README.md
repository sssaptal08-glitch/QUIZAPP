# QUIZAPP

**🎯 QuizApp – Java Based Quiz Application**

**📌 Project Overview**

QuizApp is a Java-based application designed to conduct multiple-choice quizzes in an interactive and user-friendly manner.
The application evaluates user responses, calculates scores, and displays results instantly.

This project demonstrates core Java concepts such as:

Object-Oriented Programming (OOP)

Classes & Objects

Arrays / ArrayList

Exception Handling

File Handling (if used)

Swing (if GUI-based) / Console I/O

**🚀 Features**

📝 Multiple Choice Questions (MCQs)

⏱️ Timer Support (Optional)

📊 Automatic Score Calculation

📈 Result Display with Correct Answers

💾 High Score Tracking (Optional)

🎨 Simple and Clean User Interface (Console / GUI)

**🛠️ Technologies Used**

Java (JDK 8 or above)

Swing / AWT (if GUI based)

VS Code / IntelliJ / Eclipse

Git & GitHub

**⚙️ How to Run the Project**

**🔹 Using Command Line**

Open terminal in project folder

Compile the project:

javac Login.java

Run the program:

java Main
🔹 Using IDE

Open project in IntelliJ / Eclipse / VS Code

Run Main.java

Start attempting the quiz 🎉
